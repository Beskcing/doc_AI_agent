### Task P0-2: llm_client.py Token 计数

**文件**: `src/llm_client.py`

新增 `LLMResponse` 数据类和 token 计数工具方法：

```python
from dataclasses import dataclass, field

@dataclass
class LLMResponse:
    content: str
    input_tokens: int = 0
    output_tokens: int = 0
    total_tokens: int = 0
    provider: str = ""
    model: str = ""

class LLMClient:
    def _count_tokens(self, text: str) -> int:
        """估算 token 数（中英文混合：字符数 / 1.5）"""
        if not text:
            return 0
        return max(1, round(len(text) / 1.5))

    def invoke(self, prompt: str, system_prompt: str | None = None) -> LLMResponse:
        messages = []
        if system_prompt:
            messages.append(SystemMessage(content=system_prompt))
        messages.append(HumanMessage(content=prompt))
        response = self._chat_model.invoke(messages)
        return LLMResponse(
            content=response.content,
            input_tokens=self._count_tokens(prompt + (system_prompt or "")),
            output_tokens=self._count_tokens(response.content),
            total_tokens=self._count_tokens(prompt + (system_prompt or "") + response.content),
            provider=self.provider,
            model=self.config.get_provider_config(self.provider).model,
        )
```

注意：`invoke_messages()` 和 `invoke_with_schema()` 同步改造返回 `LLMResponse`，调用方适配 `.content` 访问。

### Task P0-3: llm_client.py 流式输出

**文件**: `src/llm_client.py`

新增 `stream()` 方法：

```python
def stream(self, prompt: str, system_prompt: str | None = None):
    messages = []
    if system_prompt:
        messages.append(SystemMessage(content=system_prompt))
    messages.append(HumanMessage(content=prompt))
    for chunk in self._chat_model.stream(messages):
        yield chunk.content
```

### Task P3-2: 限流中间件

**文件**: `src/api/main.py`

IP 级基本限流，100 请求/分钟：

```python
from collections import defaultdict
import time

_rate_limit_store: dict[str, list[float]] = defaultdict(list)
RATE_LIMIT = 100  # requests
RATE_WINDOW = 60  # seconds

@app.middleware("http")
async def rate_limit_middleware(request: Request, call_next):
    client_ip = request.client.host if request.client else "unknown"
    now = time.time()
    window = _rate_limit_store[client_ip]
    # 清理过期记录
    _rate_limit_store[client_ip] = [t for t in window if now - t < RATE_WINDOW]
    if len(_rate_limit_store[client_ip]) >= RATE_LIMIT:
        return JSONResponse(status_code=429, content={"code": 429, "message": "请求过于频繁"})
    _rate_limit_store[client_ip].append(now)
    return await call_next(request)
```

## 风险评估

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| `invoke()` 返回类型从 `str` 变为 `LLMResponse` 导致调用方编译错误 | 管线中断 | 保持 `content` 属性访问兼容，全局搜索 `.content` 确认 |
| Alembic 首次迁移可能遗漏现有表 | 升级失败 | 先用 `alembic revision --autogenerate` 预览 SQL |
| 限流误伤正常用户 | 体验下降 | 默认 100 req/min 足够宽松，可配置 |
| Docker 构建镜像体积过大 | 部署慢 | `.dockerignore` 排除不必要目录 |
