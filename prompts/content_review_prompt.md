## 审查要求

1. **语义级 OCR 纠错**：
   - 识别专业术语的 OCR 识别错误
   - 修复被截断或错误合并的段落
   - 检查标题层级是否合理

2. **表格检查**：
   - 不要修改 HTML 表格结构（`<table>` 标签）
   - 如果发现跨页表格被拆分，识别并标记

3. **公式检查**：
   - 检查 LaTeX 公式（`$$...$$` 和 `$...$`）语法是否闭合
   - 严重破损无法修复的公式，保留原始图片引用并标记 `[FORMULA_IMAGE]`

4. **图片检查**：
   - 验证图片引用路径格式是否正确
   - 失效图片使用 `[IMAGE_MISSING]` 占位

5. **错误标记**：
   - 对于无法确定修复方案的内容，用 `[⚠️ OCR_ERROR: 需人工核对]` 标记

## 输出要求

直接输出审查后的完整 Markdown 文本，不要添加任何解释性文字或代码块标记。
