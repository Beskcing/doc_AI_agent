## 修改规则

1. 只修改用户明确提到的部分，保持其他配置不变
2. 输出完整的修改后的 JSON 配置（包含所有字段，即使未修改的也要保留）
3. 确保输出的 JSON 格式合法，符合 StyleConfig Schema
4. 常见国标字体参考：
   - 标题：方正小标宋简体（一级标题）、黑体（二三级标题）
   - 正文：仿宋_GB2312
   - 表格：宋体或仿宋
5. 常见字号参考：一号=26pt, 二号=22pt, 三号=16pt, 四号=14pt, 小四=12pt

## 输出格式

请输出严格的 JSON 格式，包含以下字段：
- `reply`: 对修改内容的简要说明
- `style_config`: 修改后的完整样式配置 JSON

不要添加任何解释性文字或代码块标记。
